## Contacts App

This is a simple Contacts app built on react-native.

To run this app

1. Install eas-cli

```bash
npm install -g eas-cli
```

2. Login to youe Expo Account

```bash
eas login
```

3. To Build APK

```bash
eas build -p android --profile preview
```

4. To Build AAB 

```bash
eas build --profile production --platform android
```
